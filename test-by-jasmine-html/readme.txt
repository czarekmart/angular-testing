Steps:

1. Install:
    bower install angular -S
    bower install angular-mocks -S
2. Download Jasmine Standalone zip from https://github.com/jasmine/jasmine/releases
3. Unzip lib directory into project folder
4. Copy SpecRunner.html to project folder
5. Edit SpecRunner.html to include your source files and test spec files.
6. Run SpecRunneer.html
