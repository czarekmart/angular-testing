1. Install:
-----------

npm install -g grunt-cli

npm init

npm install grunt grunt-contrib-connect grunt-karma karma karma-chrome-launcher -save-dev

npm install karma-jasmine load-grunt-tasks -save-dev

npm install grunt-protractor-runner grunt-protractor-webdriver -save-dev


2. Configure gruntfile.js

3. Run: grunt unitTest
    where unitTest is a name of registered task from gruntfile.js
